CS50 Section 8

HTML/CSS/JS


In today's section, we'll cover ways to incorporate media and have interesting features via four pages, "pandas," "pokemon," "the voice," and "games."


  • In pandas, we'll introduce the use of tables and images.
  • In pokemon, we'll use a Bootstrap element, carousel, to make our images appear in a very ~cool~ way.
  • In the voice, we'll learn to incorporate audio into our webpage.
  • In games, we'll learn how to incorporate javascript into our webpage and make it interactive. We'll learn how to write some javascript as well!
  • In all of these, we'll learn how to create our own CSS page to make the design nice!
